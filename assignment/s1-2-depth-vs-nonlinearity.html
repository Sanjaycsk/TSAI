What's happening?

In plain words

What's the trap?

"More layers = smarter." It feels obvious — and it's false if the layers are plain linear ones. Stacking straight maps just gives a bigger straight map.

Why do 5 = 1?

Each layer multiplies by a matrix. Matrix multiplication is associative, so W₅·W₄·W₃·W₂·W₁ can be pre-multiplied into one matrix. Five layers, one line.

So A and B tie?

Always. They're the same function class — a straight fence. On the ring both stall near a coin-flip; on the blobs both ace it. Depth changed nothing.

What does the bend add?

An activation folds the space (ReLU = a crease, GELU/Swish = a smooth bend). A fold can't be absorbed into the next matrix — so the collapse is broken and the fence can curve.

The bonus that nails it

The panel above multiplies net B's five matrices numerically and shows the product is one 1×2 row — and that it gives the exact same output as the full forward pass.

The one takeaway

Depth only buys power when a nonlinearity sits between the layers. Without it, "deep" is a costume. This is why every real net has activations.
